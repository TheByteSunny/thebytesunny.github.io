Lua 5.1 – Tutorial for Beginners

Lua is a lightweight and simple scripting language designed to be easy to learn, fast to execute, and flexible to use.

What is Lua?

Lua is widely used in games, mods, engines, and embedded systems. It is the language used by projects such as HL2SBR and HL2SB++.

Key Features

  • Dynamically typed
  • Garbage collected
  • Fast and lightweight
  • Easy to embed into applications

Functions

    return a + b
end

Tables

player.name = "Gordon"

Loops

for

for i = 1, 5 do
    print(i)
end

while

local i = 1

while i <= 5 do
    print(i)
    i = i + 1
end
